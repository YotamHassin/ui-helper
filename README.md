## Global

### Installation-Global
```sh
npm install node-helper -g
```

### Usage-Global
![help](https://github.com/YotamHassin/ui-helper/blob/master/assets/help.PNG?raw=true)

```sh
ui
```
Or
```sh
node-helper
```

Or
![help](https://github.com/YotamHassin/ui-helper/blob/master/assets/help.PNG?raw=true)
```sh
node-helper -h
```
## Local

### Installation-Local
```sh
cd <my_location>
npm install node-helper --save
```

### Usage-Local

```sh
node <my_location>/node-helper/index
```
Or
```sh
cd <my_location>/node-helper
npm start
```
