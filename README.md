
# ui-helper
A Node.js module that Helper

## Global

### Installation 
```sh
npm install ui-helper -g
```

### Usage

```sh
ui-helper
```
or
```sh
ui-helper -h
```
## Local

### Installation 
```sh
cd <my_location>
npm install ui-helper --save
```

### Usage

```sh
node <my_location>/ui-helper/index
```
